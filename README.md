# 다양한 상태 관리 전략으로 비교하는 Todo App

## 개요

1. props-drilling 방식을 사용한 Todo App
2. context-api 방식을 사용한 Todo App
3. useReducer 기능을 이용한 Todo App
4. zustand를 사용한 Todo App

## 기능

1. 할 일 추가, 삭제, 수정 기능
